## 🎯 What you will do today (hands-on only)

By the end of Day 2 you will have **built things**, not just talked about them:

1. **Mapped a real process** from your function (Procurement / HSE / Maintenance / Finance) using SIPOC and swim-lane notation
2. **Built three Power Automate flows** — an approval workflow, an Outlook-triggered automation, and a SharePoint/Excel integration
3. **Used AI document reading** to extract structured data from unstructured invoices, POs and HSE forms — then routed the results automatically

Day 1 taught you to *use* AI. Day 2 teaches you to *embed* AI into how work actually happens.

## ⚠️ Before you start

Yesterday's **Golden Rule** still applies:

> **If you wouldn't put it on a billboard in Blue Area, don't put it through an automation that uses public AI.**

Automation amplifies whatever you point it at — including data leaks. A flow that emails one customer is a mistake; a flow that emails 5,000 customers is a regulatory event. Read `resources/automation_red_list.md` before building anything that touches real data.
